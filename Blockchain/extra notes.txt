- install wsl ubuntu
- open the folder with wsl 
- Install NVM: curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
- Install node js: nvm install 16.17.0(nodejs version)
- nvm --version
- node --version
- install solidity+hardhat extension
- install prettier extension
- open settings.json and add, 
    "[solidity]": {
        "editor.defaultFormatter": "NomicFoundation.hardhat-solidity"
    },
    "[javascript]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
    }
- go to user settings and check format on save

- To use yarn instead of npm:
    - corepack enable
    - yarn --version

- yarn add solc // install solcjs (using solc js to compile the contracts in deploy.js)

// compile the contracts
- add below to package.json and use yarn compile to compile:
"scripts": {
    "compile": "yarn solcjs --bin --abi --include-path node_modules/ --base-path . -o . SimpleStorage.sol"
  }
